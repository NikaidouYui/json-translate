Κοινές ερωτήσεις σχετικά με το εργαλείο μετάφρασης JSON

Αυτό είναι ένα εργαλείο μετάφρασης JSON διεθνοποίησης που υποστηρίζεται από AI και βοηθά τους προγραμματιστές να μεταφράσουν γρήγορα τα αρχεία γλώσσας JSON σε πολλές γλώσσες διατηρώντας την ακεραιότητα της δομής JSON.

1. Ανεβάστε το αρχείο JSON σας για μετάφραση 2. Επιλέξτε τις γλώσσες στόχο 3. Εισάγετε το OpenAI API Key σας 4. Κάντε κλικ στην έναρξη της μετάφρασης 5. Κατεβάστε τα αποτελέσματα μετά την ολοκλήρωση της μετάφρασης

Αυτή τη στιγμή υποστηρίζει πολλές γλώσσες, συμπεριλαμβανομένων των Αγγλικών, Κινέζικων (Απλοποιημένων), Κινέζικων (Παραδοσιακών), Ιαπωνικών, Κορεατικών, Γαλλικών, Γερμανικών, Ισπανικών, Ρωσικών και άλλων.

Όχι. Το API Key σας χρησιμοποιείται μόνο προσωρινά στον περιηγητή και δεν θα αποθηκευτεί ή μεταδοθεί σε κανέναν διακομιστή.

Χρησιμοποιούμε το μοντέλο GPT της OpenAI για μετάφραση, διασφαλίζοντας την ακρίβεια των τεχνικών όρων και την έκφραση φυσικής γλώσσας. Υποστηρίζουμε επίσης τη λειτουργικότητα προσαρμοσμένου λεξικού για τη διατήρηση της συνέπειας συγκεκριμένης ορολογίας.

Το μέγεθος ενός αρχείου JSON περιορίζεται στα 10MB. Για μεγαλύτερα αρχεία, συνιστάται να τα χωρίσετε και να τα μεταφράσετε σε παρτίδες.
