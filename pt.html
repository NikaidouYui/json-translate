Perguntas comuns sobre a ferramenta de tradução JSON

Esta é uma ferramenta de tradução de internacionalização JSON alimentada por IA que ajuda os desenvolvedores a traduzir rapidamente arquivos de linguagem JSON em vários idiomas, mantendo a integridade da estrutura JSON.

1. Carregue seu arquivo JSON para tradução 2. Selecione os idiomas de destino 3. Insira sua chave de API OpenAI 4. Clique em iniciar tradução 5. Baixe os resultados após a conclusão da tradução

Atualmente suporta vários idiomas, incluindo inglês, chinês (simplificado), chinês (tradicional), japonês, coreano, francês, alemão, espanhol, russo e mais.

Não. Seu API Key é usado apenas temporariamente no navegador e não será salvo ou transmitido para nenhum servidor.

Usamos o modelo GPT da OpenAI para tradução, garantindo a precisão dos termos técnicos e a expressão em linguagem natural. Também suportamos a funcionalidade de dicionário personalizado para manter a consistência da terminologia específica.

O tamanho de um único arquivo JSON é limitado a 10MB. Para arquivos maiores, é recomendável dividi-los e traduzir em lotes.
