Domande comuni sullo strumento di traduzione JSON

Questo è uno strumento di traduzione JSON basato su IA che aiuta gli sviluppatori a tradurre rapidamente i file di lingua JSON in più lingue mantenendo l'integrità della struttura JSON.

1. Carica il tuo file JSON per la traduzione 2. Seleziona le lingue di destinazione 3. Inserisci la tua Chiave API OpenAI 4. Clicca avvia traduzione 5. Scarica i risultati dopo il completamento della traduzione

Attualmente supporta più lingue tra cui inglese, cinese (semplificato), cinese (tradizionale), giapponese, coreano, francese, tedesco, spagnolo, russo e altre.

No. La tua Chiave API viene utilizzata solo temporaneamente nel browser e non verrà mai salvata o trasmessa a nessun server.

Utilizziamo il modello GPT di OpenAI per la traduzione, garantendo l'accuratezza dei termini tecnici e l'espressione naturale. Supportiamo anche la funzionalità di dizionario personalizzato per mantenere la coerenza della terminologia specifica.

La dimensione del singolo file JSON è limitata a 10MB. Per file più grandi, si consiglia di dividerli e tradurli in batch.
